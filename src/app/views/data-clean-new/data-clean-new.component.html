Batch Clean

  • There is no interface on this page for you to process data, as it has to be running locally
  • Due to the limiation of pure frontend project, it involves some manual steps to work with mutiple files
  • The limitations are:
  • No generic way of loading all source data files, or it makes it too heavy to have all files stored in frontend (18.76GB)
  • The limitaion of computing ability of frontend project, as everything happens in frontend, it takes about 10 minutes to generate a 43000 records clean data
  • Manual steps are:
  • Running this page locally, feed source file of 1 month data for each round
  • This batch clean is using the same mechanism as the single file clean, but filtering out every minute data instead of every second, for a faster processing speed
  • Possible future work for this tool is to setup the backend server and store the file in some cloud storage account, to have a fast processing speed and light project
